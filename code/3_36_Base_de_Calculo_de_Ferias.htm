3.36 Base de Cálculo de Férias

Deverá estabelecer o mesmo critério para o cálculo da base do 13º salário. A diferença está em que a base do 13º salário será levantada no período correspondente ao ano civil e a base de férias, durante o período aquisitivo das mesmas.
