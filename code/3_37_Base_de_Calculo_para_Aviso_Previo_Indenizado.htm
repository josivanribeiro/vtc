3.37 Base de Cálculo para Aviso Prévio Indenizado

A base de cálculo do aviso prévio indenizado será a maior remuneração recebida durante o último período. Ou seja, a partir da data do aviso prévio retroagir um ano, e comprovar a maior remuneração recebida.
