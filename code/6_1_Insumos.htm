6.1 Insumos

Insumos são todos os itens que entram no processo de produção de mercadorias ou serviços. Como entram no processo produtivo de serviços? Todos os desembolsos efetuados para a realização do objeto prestado (serviço). A essência (máquina) para a realização dos serviços está no trabalhador treinado, uniformizado, seguro, assistido, com os seus direitos respeitados. Para atingir o objetivo ele necessita de equipamentos, ferramentas, estrutura e materiais para atingir os objetivos. Todos os dispêndios que entram neste processo de produção são considerados insumos Para que possamos elaborar nossa proposta de preços a mesma deve, como já frisamos, anteriormente, constituir todos os custos necessários para a sua realização.
Devemos verificar todo o processo e a legislação para que possamos determinar nos insumos os custos de cada item componente.
