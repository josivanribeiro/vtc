7.3.3 Capachos de Fibra Sintética, Internos

Limpar com aspirador de pó.
Lavar, semanalmente, com jato, água e sabão neutro. Colocar para escorrer o excesso de água. Em época de chuvas este cuidado deve ser aplicado com maior freqüência. Aconselha-se a efetuar esta lavagem antes de finais de semana ou feriados.
